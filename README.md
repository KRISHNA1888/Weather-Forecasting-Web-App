# Weather Forecasting Web App

A responsive weather forecasting web application that provides real-time weather information and a 7-day forecast using the OpenWeatherMap API.

## Features
- Search weather by city
- Use current location
- Display temperature, humidity, wind speed
- Show sunrise and sunset time
- 7-day weather forecast
- Responsive design for desktop, tablet, and mobile

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
